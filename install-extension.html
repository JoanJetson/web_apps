How to Install and Use VS Code's Live Server Extension

Install | Use | Video

Installation Instructions

  1. In Visual Studio Code, click the Extesnions icon in the left column toolbar
  2. In the search bar that appears, type Live Serve.
  3. Select the Live the Server Extension. I should be the top choice
  4. In the page for Live Server that appears, click the green Install Button.

Usage Instructions

  • Visual Studio Code must be open to a folder to use live server.
  • In VS Code's Explorer, right click (Windows) or Control+click (macOS) on an html file. From the content menu that appears, select Open with live server.
